
Create a modern, professional Typing Practice Web App using HTML, CSS, and JavaScript.

Design Requirements:
- Clean, minimal, modern UI
- Dark + Neon color scheme (black background with glowing blue/purple accents)
- Smooth animations and transitions
- Fully responsive (mobile + desktop)
- Realistic typing experience

Main Sections:

1. HERO / START SECTION
- Title: "Typing Master Pro"
- Subtitle: "Improve your typing speed & accuracy"
- Start Button (big, glowing)
- When clicked → scroll to typing test

2. MODE SELECTION
- Difficulty Levels:
  - Easy
  - Medium
  - Hard

- Time Options:
  - 30 seconds
  - 60 seconds

- Selected option should be highlighted

3. TYPING TEST SECTION
- Display a paragraph based on difficulty:
  - Easy: simple words
  - Medium: normal sentences
  - Hard: complex words & punctuation

- Input box where user types
- Real-time features:
  - Correct letters → green
  - Wrong letters → red underline
  - Current letter highlight
  - Live WPM counter
  - Live accuracy %

- Timer countdown visible

4. LIVE FEEDBACK SYSTEM
- Mistakes count (real-time)
- Accuracy percentage
- Words per minute (WPM)
- Highlight incorrect characters instantly

5. RESULT SECTION (After time ends)
- Final WPM
- Accuracy %
- Total mistakes
- Performance level:
  - Beginner
  - Intermediate
  - Pro

- Restart button

6. TRAINING & SKILLS SECTION
Add a separate section with 3–4 paragraphs explaining:
- Benefits of typing speed
- How to improve typing
- Practice tips
- Why accuracy matters

7. EXTRA FEATURES
- Sound effect on key press (optional)
- Restart test anytime
- Smooth scroll between sections
- Keyboard focus auto start

8. TECH REQUIREMENTS
- Pure HTML, CSS, JS (no frameworks)
- Well-structured code
- Separate files:
  - index.html
  - style.css
  - script.js

9. UX DETAILS
- One-click start typing
- Auto-focus input field
- Real-time cursor tracking
- Clean typography (Google Fonts style)

10. VISUAL STYLE
- Glassmorphism + neon glow
- Rounded corners
- Soft shadows
- Modern UI like premium apps

Goal:
Build a fully working typing practice website with real-time feedback, modern UI, and smooth user experience that feels like a professional product.
